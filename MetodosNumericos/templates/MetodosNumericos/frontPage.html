Método de Bisección

El método de bisección es un algoritmo de búsqueda de raíces que trabaja dividiendo el intervalo a la mitad y seleccionando el subintervalo que contiene la raíz.

Algoritmo:

  1. Seleccionar intervalo [a, b] donde f(a)*f(b) < 0
  2. Calcular punto medio c = (a + b)/2
  3. Evaluar f(c) y determinar nuevo intervalo
  4. Repetir hasta alcanzar precisión deseada

Eliminación Gaussiana

Método para resolver sistemas de ecuaciones lineales mediante la transformación de la matriz aumentada a forma escalonada.

Pasos principales:

  1. Formar matriz aumentada [A|b]
  2. Aplicar operaciones elementales para obtener forma triangular
  3. Resolver mediante sustitución hacia atrás

Gauss-Jordan

Variante de la eliminación gaussiana que transforma la matriz a forma escalonada reducida.

Diferencias con Gauss:

  • Se obtienen unos en la diagonal principal
  • Se eliminan términos por encima y debajo del pivote
  • Resulta en solución directa sin sustitución

Interpolación de Lagrange

Método para encontrar el polinomio único de grado n-1 que pasa exactamente por n puntos dados.

Fórmula general:

P(x) = Σ yᵢ * Lᵢ(x)

donde Lᵢ(x) = Π (x - xⱼ)/(xᵢ - xⱼ) para j ≠ i

Interpolación de Newton

Método de interpolación polinómica que utiliza diferencias divididas.

Fórmula general:

Pₙ(x) = f[x₀] + f[x₀,x₁](x-x₀) + f[x₀,x₁,x₂](x-x₀)(x-x₁) + ...

Ventajas sobre Lagrange:

  • Fácil de actualizar con nuevos puntos
  • Menos operaciones computacionales
